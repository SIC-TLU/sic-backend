## Description

SIC (Student information Club) is a club with aling tradition. There are many exciting activities with the enthusiasm of the menbers, not only helping each other in studying but also in life skills.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
